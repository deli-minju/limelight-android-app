# 🎬 LimeLight Android Application

**LimeLight 영화 예매 서비스의 공식 안드로이드 클라이언트 애플리케이션입니다.**

이 앱은 [limelight-movie-booking](https://github.com/deli-minju/limelight-movie-booking) 웹 프로젝트와 연동되어 사용자에게 모바일 환경에서의 영화 예매 기능을 제공합니다.

## ✨ 주요 기능

-   영화 목록 조회

## 🛠️ 개발 환경

-   Android Studio Hedgehog | 2023.1.1
-   Java
-   Gradle 8.2
-   Target SDK: 34 (Android 14)
